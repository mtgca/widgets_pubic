Hierarchical Clustering Linkage Methods

Summary of Linkage Methods

1. Single Linkage (Nearest Neighbor): Distance is defined as the shortest distance between any single point in Cluster A and any single point in Cluster B. It tends to produce long, "chain-like" clusters.
2. Complete Linkage (Farthest Neighbor): Distance is defined as the longest distance between any point in Cluster A and any point in Cluster B. It tends to produce highly compact, spherical clusters.
3. Average Linkage (UPGMA): Calculates the distance between all possible pairs across the two clusters and takes the mean average. A robust middle-ground approach.
4. Centroid Linkage: Finds the geometric center (centroid) of each cluster and measures the direct distance between these two centers.
5. Ward's Method: Unlike the others, this method evaluates variance. It measures the increase in the total within-cluster variance (Error Sum of Squares) that would occur if the two clusters were merged into a single super-cluster.

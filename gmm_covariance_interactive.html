Gaussian Mixture Models: Covariance Types

Covariance Shapes

The covariance matrix defines the shape, size, and orientation of the clusters. Select a type to see how the mathematical constraints affect the GMM flexibility.

Spherical

Each component has its own single variance value. The shape is constrained to a perfect circle (or sphere in 3D). Clusters can have different sizes, but their axes must be equal.

Diagonal (Diag)

Each component has an independent variance for each dimension, but covariances are zero. Shapes are ellipses constrained to align with the coordinate axes (no tilted rotation).

Tied

All components share the exact same general covariance matrix. This forces all clusters to have the exact same shape, size, and orientation, varying only by their mean (center).

Full

Each component has its own independent, general covariance matrix. This provides maximum flexibility, allowing ellipses of any size and any arbitrary orientation.

Recurrent Neural Networks: Unrolling the Computational Graph

An RNN is one cell with a feedback loop: its hidden state ht is fed back as input to the next time step, acting as a memory of everything seen so far. Unrolling that loop over time turns it into a chain — one copy of the same cell per time step. Step through the unrolling below.

🧠 Recurrence and memory

The network has a feedback loop: the hidden state ht is an internal memory of past inputs.

ht summarizes the sequence history up to step t — the whole past, compressed into one vector.

⛓️ Sequential dependence

Unlike a feed-forward network, the state at time t depends on both the current input xt and the previous context ht−1.

Order matters: feeding the same inputs in a different order produces different states.

♻️ Parameter sharing

The same functions f and g (and their underlying parameters) are reused at every time step.

This lets the model process sequences of arbitrary length with a fixed number of parameters — unrolling adds time steps, never new weights.
